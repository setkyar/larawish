Installation
------------

Add the package-skeleton package to your `composer.json` file.

``` json
{
    "require": {
        "setkyar/larawish": ""
    }
}
```

Or via the command line in the root of your Laravel installation.

``` bash
$ composer require "setkyar/larawish"
```
